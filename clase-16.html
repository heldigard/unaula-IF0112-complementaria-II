Requisitos minimos

Hardware

  • Microcontrolador (ESP32 o similar).
  • Minimo 2 sensores diferentes.
  • Minimo 1 actuador.
  • Circuito de alimentacion apropiado.

Software

  • Lectura de sensores funcionando.
  • Logica de decision implementada.
  • Actuador controlado segun la logica.
  • Conectividad WiFi activa.
  • Comunicacion con nube/servidor.

Conectividad

  • Protocolo de comunicacion (MQTT/HTTP).
  • Interfaz web o app movil.
  • Visualizacion de datos en tiempo real.

Documentacion

  • Diagrama de arquitectura.
  • Diagrama de conexiones fisicas.
  • Codigo comentado.
  • Manual de usuario.
  • Video demostrativo.

Ejemplos de proyectos exitosos

Proyecto 1: Sistema de riego inteligente

  • ESP32 + DHT22 + sensor humedad + bomba.
  • App Blynk con modo automatico y manual.
  • Ahorro 40% de agua vs riego manual.
# Fragmento de codigo (Blynk)
void checkHumedad() {
  int humedad = analogRead(HUMEDAD_PIN);
  humedad = map(humedad, 0, 4095, 0, 100);

  if (humedad < 30) {
    digitalWrite(BOMBA_PIN, HIGH);
    Blynk.notify("Riego activado");
  } else if (humedad > 60) {
    digitalWrite(BOMBA_PIN, LOW);
  }
}

Proyecto 2: Monitor de calidad del aire

  • ESP32 + PMS5003 + SCD30 + OLED.

  • Autonomia 8h con bateria.

Proyecto 3: Cerradura inteligente

Proyecto 4: Seguimiento de mascotas
